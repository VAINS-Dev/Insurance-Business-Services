Description: The SurrenderQuote web service provides the values for a surrender of the policy such as cash value, fund value, dividends, withholding, and outstanding loans based on company code, policy number, and effective date.

Modified in- V20 SP1

Contract Changes - 1. Added TefraGross field in TefraTaxValuesFields object and TaxValuesData , GMSVValue objects and LoanAllowed field added in Response class. 2. Added ArrearsFeeOverride, DoTaxRulesApplyOverride, ETIFaceAmountIn, ETIRPUAmtToLeaveInSusp, FederalWithholdingOverride, FederalWithholdingOverrideAmount, FederalWithholdingOverridePctAmt, GenerateConfirm, MVAOverride, ProcessingFee, RPUPayoffLoanFlag, RequoteSwitch, StateWithholdingOverride, StateWithholdingOverrideAmount, StateWithholdingOverridePctAmt, SuppressCheck, SurrenderLoadOverride and SurrenderLoadOverrideAmount fields in request.
3. Added BaseFaceAmt, ETIActiveOrAmt, ETIAmtLeaveSus, ETIFaceAmt, TotalAmt, PurchaseETIAmt, Expiration, RPUActiveOrAmt, RPUAmtLeaveSus, RPUFaceAmt, TotalSurrenderAmt, PurchaseRpuAmt,ContractualFree, Earnings, FreePurchasePay and DepositHistoryInfo fields in response class. 4. Added ShadowBenefitVL and ShadowBenefitValues class added in the Response class with fields Names SBVLBaseFundValue,SBVLFreeWithdrawl,SBVLHighestBen,SBVLNetFundValue,SBVLShadowAdjust,SBVLSurChg and ShadowBenefitSeq,ShadowBenefitID,ShadowBenefitFundValue 5. Added UserType and CoderID field in header. 6. Removed GUID from request body and added in route.
